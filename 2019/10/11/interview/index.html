  • Axios的特点有哪些
    1、在浏览器中发送 XMLHttpRequests 请求；
    2、在 node.js 中发送 http请求；
    3、基于 Promise 的 HTTP 请求客户端,支持 Promise API；
    4、拦截请求和响应；
    5、转换请求和响应数据；
    6、自动转换 JSON 数据；
    7、客户端支持保护安全免受 XSRF 攻击；

    第二种：组件内的钩子
    第三种：单独路由独享组件
